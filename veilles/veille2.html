1. Contexte & Enjeux Réseau

Un stade moderne est une plateforme informatique à forte densité humaine : jusqu'à 90 000 personnes connectées simultanément, chacune avec un ou plusieurs appareils. Ce défi de connectivité est unique et implique de gérer plusieurs réseaux en parallèle.

⚠️ Contraintes techniques

  • Densité extrême : jusqu'à 1 utilisateur/m² en tribunes.
  • Pics synchronisés : 80% du trafic concentré en 5 min (but, mi-temps).
  • Interférences RF : structures métalliques perturbant la propagation radio.

📡 Multi-réseaux coexistants

  • Guest Wi-Fi : fans, streaming, réseaux sociaux.
  • Opérationnel : caisses, CCTV, contrôle d'accès.
  • Broadcast : flux vidéo HD vers les régies TV.
  • Sécurité : TETRA, pompiers, services médicaux.
📌 Exigence FIFA : Disponibilité réseau de 99,999% — soit moins de 5 min d'interruption par an — conforme aux standards datacenter Tier IV. L'Allianz Arena et le Santiago Bernabéu disposent de leur propre datacenter on-site pour y répondre.

2. Wi-Fi Haute Densité — Architecture Pico Cell

Dans un stade, le Wi-Fi classique (peu de bornes puissantes) s'effondre sous la charge. La solution : déployer de nombreuses bornes à faible puissance sous chaque rangée de sièges — c'est l'architecture Pico Cell.

Exemple — Tottenham Hotspur Stadium : 1 641 points d'accès HPE Aruba, soit 1 borne pour 38 fans. Chaque AP dessert 30 à 75 personnes, garantissant un débit stable même lors des pics de trafic.

Protocoles clés

Roaming & Load Balancing

  • 802.11r : handover entre bornes en < 50 ms, sans ré-authentification.
  • 802.11k & 802.11v : le réseau suggère proactivement une borne moins chargée à l'appareil (load balancing automatique).

Wi-Fi 6 (802.11ax)

  • OFDMA : sert plusieurs appareils simultanément sur le même canal.
  • BSS Coloring : bornes adjacentes utilisent le même canal sans se brouiller.
  • Gains mesurés : 4× à 10× en débit vs Wi-Fi 5 en environnement dense.

3. DAS & 5G Privée

Le Wi-Fi couvre les connexions volontaires, mais les fans veulent aussi un signal mobile (4G/5G) fort sans agir. C'est le rôle du DAS (Distributed Antenna System) : un réseau d'antennes réparties dans tout le bâtiment, reliées à une tête de réseau centrale.

Modèle Neutral Host (Tottenham) : Une seule infrastructure DAS gérée par Shared Access Limited accueille les 4 opérateurs britanniques (Vodafone, O2, EE, Three) simultanément — évitant de multiplier les installations par opérateur.

5G Privée

  • Fréquence licenciée dédiée, isolée du réseau public.
  • Latence garantie < 5 ms pour les applications temps réel.
  • Orange Vélodrome : 1er stade 5G de France (2019). Test Sony : envoi de photos HD directement via 5G sans câble.

Network Slicing

  • Slice Sécurité : priorité max pour les urgences.
  • Slice Broadcast : débit garanti pour les flux TV 4K.
  • Slice Fan : bande passante partagée pour le public.
  • Slice IoT : milliers de capteurs à faible débit.

4. Architecture Réseau & VLANs

La segmentation par VLANs isole les flux hétérogènes d'un stade (billetterie, CCTV, fans, broadcast…) pour garantir sécurité et performance. L'architecture Cisco Connected Stadium repose sur 3 couches : Core, Distribution, Accès.

6. Cybersécurité des Stades

🎯 Surfaces d'attaque

  • Wi-Fi Guest : Evil Twin, Man-in-the-Middle.
  • POS : interception données CB (skimming réseau).
  • Caméras IP : point d'entrée pour mouvement latéral.
  • DDoS : saturation du contrôle d'accès pour bloquer l'entrée.
  • Ransomware : chiffrement des scoreboards pendant l'événement.

🛡️ Contre-mesures

  • VLAN strict : Guest ne peut jamais accéder à POS ou CCTV.
  • Portail captif : authentification avant accès Wi-Fi fan.
  • IDS/IPS : détection comportements anormaux en temps réel.
  • 802.1X : certificat obligatoire pour le réseau opérationnel.
  • SIEM centralisé : corrélation des logs de tous les équipements.
Euro 2016 : Tous les stades français reliés à un Media Data Center unique via fibre optique. Des équipes SOC ont assuré la surveillance 24h/24 pendant toute la compétition, protégeant un datacenter central stratégique diffusant les images TV au monde entier.

7. Edge Computing, IA & Perspectives

Edge Computing : un mini-datacenter on-site traite les données critiques localement (contrôle d'accès, CCTV, gestion de foule) sans passer par le Cloud, réduisant la latence à < 5 ms. Seules les données agrégées remontent vers le Cloud.

🎥 IA & Gestion des foules

  • Computer vision sur les flux CCTV pour détecter les zones de surpopulation.
  • Alertes automatiques aux agents de sécurité en cas de comportement anormal.
  • Heatmap temps réel des déplacements de supporters.

📊 Data Analytics

  • Analyse des parcours fans via le Wi-Fi pour optimiser les buvettes.
  • Push notifications personnalisées selon la position dans le stade.
  • Allianz Arena : SAP HANA pour consolider toutes ces données en temps réel.

Horizon 2027

Wi-Fi 7 (802.11be) — 46 Gbit/s théoriques, Multi-Link Operation pour combiner plusieurs bandes simultanément.
5G SA & Network Slicing généralisé — tranches réseau dédiées par type d'événement (concert, sport, e-sport).
Réalité Augmentée en tribunes — stats joueurs en temps réel via lunettes AR, nécessitant < 10 ms de latence (5G mmWave).
